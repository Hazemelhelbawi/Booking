# Booking App

A React app for booking doctor appointments.

## Features

- User sign up & login
- Browse doctors by specialty
- Book appointments with time slots
- Edit user profile & picture
- Responsive design

## Tech

- React, React Router
- Context API
- Tailwind CSS
- react-hot-toast

## Usage

```sh
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
